# Snake Game

A classic Snake game implementation in Python using Pygame.

## Requirements

- Python 3.7+
- Pygame

## Installation

1. Install the required dependencies:
```bash
pip install -r requirements.txt
```

## Running the Game

```bash
python snake.py
```

## Controls

- **Arrow Keys**: Move the snake (Up, Down, Left, Right)
- **ESC**: Quit the game
- **SPACE**: Pause/Resume the game

## Game Rules

- Guide the snake to eat food (red squares)
- Each food eaten increases your score by 10
- The snake grows by 1 segment with each food consumed
- Avoid hitting the walls or yourself
- Game ends when you collide with the walls or your own body

## Game Features

- Score tracking
- Pause/Resume functionality
- Increasing difficulty as the snake grows
- Smooth controls with directional input
